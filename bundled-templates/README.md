# Bundled Templates

Place `.ektmp` template files in this folder to distribute them with the app.

**Workflow:**

1. Design your template visually in the app
2. Export it as `.ektmp` (right-click → Export)
3. Copy the `.ektmp` file into this folder
4. Build the app — templates will be included in the installer

Templates placed here are automatically copied to the user's templates directory on first launch (or when new templates are added in updates).
